Virutal Scroll

Virtual Scroll displays a virtual, "infinite" list.

npm install angular2-virtual-scroll

What is it?

The problem: Let’s say you have a million rows of data, and to simply create a million dom nodes and placing them to your html document’s body is a good way to freeze or crash your browser. Remember only a portion of that data is viewable to the user at any time and rest of them are accessed through scroll.

The solution: Solution is to place the nodes that are only viewable to the user in the dom and modify them as the user scrolls.

The approach: This module will let you to place an array of records and define a template to render each item in the array: headers, title, icons and footers etc. Not every record in the list is rendered at once; instead a small subset of records (enough to fill the viewport) are rendered and reused as the user scrolls.

Virutal Scroll

Virtual Scroll displays a virtual, "infinite" list.

npm install angular2-virtual-scroll

About

This module does not render every record in the list at once; instead a small subset of records just enough to fill the viewport are rendered and reused as the user scrolls. Thus display an infinitely growing list of items in a viewport of size just about a couple rows in an effcient way is made possible.

  • Angular 2 compatible module
  • Easy to use apis

Usage


    <virtual-scroll [items]="items" (update)="viewPortItems = $event">

        <list-item *ngFor="let item of viewPortItems" [item]="item">
        </list-item>

    </virtual-scroll>
